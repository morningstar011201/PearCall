# GhostCall

Instant P2P voice calls. No signup. No contacts. Just a link.

🌐 Live at [ghostcall.space](https://ghostcall.space)

## What it is
A zero-signup voice calling app. Share a link or 6-digit code — connect instantly from any browser.

## Features
- No account required
- Peer-to-peer audio (WebRTC)
- Join by link or 6-digit code
- Calls vanish when you hang up
- Call quality indicator
- Auto copy link
- Works on all devices

## Tech Stack
- WebRTC + PeerJS
- Cloudflare TURN Server
- Supabase (analytics)
- Vercel (hosting)

## Files
- `index.html` — Landing page
- `app.html` — Calling app
- `dashboard.html` — Admin analytics
- `sitemap.xml` — SEO sitemap
- `robots.txt` — SEO robots
- `vercel.json` — Routing + security headers
